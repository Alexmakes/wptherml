Installation Instructions
=========================


from github::

    git clone https://github.com/FoleyLab/wptherml.git
    cd wptherml
    python3 setup.py install


running unit tests from cloned repository::

    cd test
    python3 -m pytest test.py


from PyPI::

    pip3 install wptherml
